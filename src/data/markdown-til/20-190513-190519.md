- http와 https
  - HTTP는 서버와 클라이언트가 통신하는 통신규약이다. 서로 통신할 때 plainttext로 주고받고, 이런 정보가 전달되는 네트워크에서 전송되는 정보를 엿볼 수 있다. 그래서 서버와 클라이언트가 정보를 주고받기 전에 암호화된 통신 채널을 설정한 후에 정보를 주고받도록 했다. 이게 HTTPS(HTTP Secure). http 메세지를 암호화하는 것.
